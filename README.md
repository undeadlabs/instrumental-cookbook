# instrumental-cookbook

Installs and configures [Instrumental](http://instrumentalapp.com)

## Supported Platforms

* Ubuntu

## Usage

### instrumental::default

Include `instrumental` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[instrumental::default]"
  ]
}
```
